# MediConnect AI CRM

AI-powered Healthcare CRM built using React, FastAPI, LangGraph, and Groq LLM.

---

## Features

- AI-powered HCP interaction logging
- Dashboard with analytics
- Interaction history
- Follow-up tracking
- LangGraph AI workflow
- Groq LLM integration
- FastAPI backend
- React frontend
- SQLite database

---

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- FastAPI
- Python
- SQLAlchemy
- LangGraph
- Groq LLM

### Database
- SQLite

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
source venv/bin/activate
uvicorn main:app --reload
```
